# Real-Time Chat Application

A room-based real-time chat application built with Java, Spring Boot, WebSocket, STOMP, MongoDB, HTML, CSS, and JavaScript.

## Features
- Real-time messaging
- Room-based chat
- MongoDB message persistence
- Chat history loading
- Simple responsive UI

## Tech Stack
- Java 17
- Spring Boot
- WebSocket + STOMP
- MongoDB
- HTML/CSS/JavaScript

## Run MongoDB
Make sure MongoDB is running on:
mongodb://localhost:27017

## Run Application
mvn spring-boot:run

## Open
http://localhost:8080
